## Описание
Приложение «Movie Library» — это графическое приложение на Python (библиотека Tkinter) для управления личной коллекцией фильмов. Оно позволяет добавлять фильмы, просматривать их в виде таблицы, фильтровать по жанру и году выпуска, а также сохранять данные в JSON-файл.

## Функционал
- Добавление фильмов с валидацией данных (год — число, рейтинг — 0–10).
- Отображение списка фильмов в таблице (`Treeview`).
- Фильтрация:
  - По жанру (выпадающий список).
  - По году выпуска (ручной ввод).
- Автоматическое сохранение и загрузка данных через JSON.
- Версионирование через Git.

## Требования
- Python 3.8 или выше.
- Стандартные библиотеки: `tkinter`, `json`, `os`.
